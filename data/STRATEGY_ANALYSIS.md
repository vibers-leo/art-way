# Art-Way (아트웨이) 전략 진단 리포트

> PM Skills 프레임워크 기반 전략 분석 | 2026-03-26

---

## 1. 프로젝트 개요 및 현재 상태
- **역할**: 아트 콘텐츠 플랫폼 — 전시 및 아카이브 관리 웹앱
- **개발 완성도**: 85% (core UX 완성, 관리자 기능 고도화 중)
- **에코시스템 내 위치**: Vibers 핵심 콘텐츠 플랫폼

## 2. 핵심 가치 & 포지셔닝
- **플랫폼 역할**: 아트 전시/콘텐츠 중앙 관리소 — BlockNote 리치 에디터 기반
- **기술 강점**: Supabase Auth SSR + shadcn/ui 디자인 시스템 (미니멀 모노크롬)
- **AI Recipe 통합**: 갤러리/아카이브 이미지 자동 보강 및 전시 홍보 포스터 생성
- **포트폴리오 가치**: 아트 콘텐츠 플랫폼 아키텍처 레퍼런스 (에디터+Auth+Storage)

## 3. 주요 리스크 & 과제
1. **사용자 확보**: 아티스트/전시 주최자 유입 메커니즘 부재
2. **수익화 모델 미정**: 플랫폼으로서의 지속 가능성 전략 필요
3. **콘텐츠 모더레이션**: 사용자 생성 콘텐츠 관리 정책 미수립

## 4. 권장 액션
- **단기(1개월)**: 관리자 대시보드 UI 정리, 권한 관리(read/write) 강화
- **중기(3개월)**: 아티스트 커뮤니티 온보딩 프로그램, 기본 수익화 모델(광고/premium features) 설계

## 핵심 한 줄 요약
아트 콘텐츠 중앙 관리 플랫폼 — BlockNote 에디터와 AI 이미지 통합으로 전시/아카이브 생산성 극대화, 향후 아티스트 커뮤니티 플랫폼으로의 전환 가능성.
